# Play with Array

## Def  Array

```text
games = ["csgo", "dota2", "lol"]
puts games
```

![](../.gitbook/assets/image%20%2863%29.png)

## Loop Array

```text
#方法1
games.each do |game|
    puts "我爱#{game}"
end
#方法2
games.each_with_index do |game,i|
	puts "#{i+1}.#{game}"
end
```

![](../.gitbook/assets/image%20%2840%29.png)

## Combine Array

```text
puts games.join("-")
```

![](../.gitbook/assets/image%20%2851%29.png)
