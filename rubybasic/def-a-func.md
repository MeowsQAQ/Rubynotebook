# Def a func

## def a simple func

函数以`def`开头，以`end`结尾，def后紧跟函数名称，中间为函数内容

```text
def sayHello
	puts"helloworld meows"
end
sayHello #执行函数
```

## def a func and feed a param

```text
def sayHelo(name="meow")
	puts"helo #{name}"
end
# 未指定参数时使用默认值
sayHelo
sayHelo("meows")
```
