# Play with operational character

* 运算操作符
  * +：加法
  * -：减法
  * \*：乘法
  * \*\*：阶乘
  * /：除法
  * %：取余
* 比较操作符
  * ==：等于
  * !=：不等于
  * ＞：大于
  * ＞=：大于等于
  * ＜：小于
  * ＜=：小于等于
* 逻辑操作符
  * &&：逻辑与
  * \|\|：逻辑或
  * ！：逻辑非
* 位运算操作符\(不常用\)
  * &：与AND
  * \|：或OR
  * ^：亦或
  * ~：位反转
  * ＜＜：左移位
  * ＞＞：右移位

```text
 puts 2**3
 puts 10%3
 if 1!=2 then
 	 puts "1!=2"
 else
 	 puts "1==2"
 end 
 a=5
 b=10
 if a<=b then puts "ok" end
 b=5
 if a<=b then puts "ok" end
```
